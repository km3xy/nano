GNU nano 是 Linux / Termux 里最简单的文本编辑器之一，非常适合你在 Termux 里写脚本（比如 Node.js、PHP 等）。 下面是 最实用的 nano 使用方法 👇 一、打开或创建文件 Bash nano 文件名 例如： Bash nano test.js 作用： 如果文件存在 → 打开编辑 如果文件不存在 → 自动创建 二、编辑内容 打开后你就可以 直接输入文字。 例如写 Node.js： console.log("Hello World"); nano 不需要进入“编辑模式”，打开就可以直接输入。 三、保存文件 按： Ctrl + O (注意不要点+号 点Ctrl后，直接点大写的O) 屏幕下面会显示： File Name to Write: test.js 然后按： Enter 就保存成功。 四、退出 nano 按： (点击Ctrl后按键变成选中色，然后不要点+号直接点大写的X) Ctrl + X (注意不要点+号 点Ctrl后，直接点大写的X) 如果修改过文件，会提示： Save modified buffer? Y Yes N No 按： Y 然后 Enter。 五、删除一行 把光标放在那一行： Ctrl + K (注意不要点+号 点Ctrl后，直接点大写的K) 整行删除。 六、复制一行 Alt + 6 七、粘贴 Ctrl + U 八、搜索文字 Ctrl + W 输入要搜索的内容： console 然后 Enter。 九、跳到指定行 Ctrl + _ 输入行号： 20 十、Termux手机快捷键 手机没有键盘时： 长按 Termux屏幕 → 打开菜单 → Keyboard 或者用： 音量下 + 字母 例如： 音量下 + O = Ctrl + O 音量下 + X = Ctrl + X 音量下 + K = Ctrl + K 一个完整示例 创建 Node.js 文件： Bash nano test.js 输入： console.log("Termux Node.js OK"); 保存： Ctrl + O Enter Ctrl + X 运行： Bash node test.js 输出： p Termux Node.js OK 如果你愿意，我可以再教你 3个Termux最常用编辑技巧： 1️⃣ nano 快速写 1000 行代码 2️⃣ nano 自动补全 3️⃣ nano 一键运行脚本 学会后 手机写程序速度会快 5 倍。 你发的 123 👍 我给你演示一个 最简单的 nano 实操例子，在 Termux 里一步一步做。 1️⃣ 打开 nano 创建文件 输入： Bash 复制代码 nano test.txt 屏幕会进入 GNU nano 编辑界面。 2️⃣ 输入内容 在里面直接输入： 复制代码 123 你好 Termux nano 不用进入编辑模式，直接打字就可以。 3️⃣ 保存文件 按： 复制代码 Ctrl + O 手机没有 Ctrl 可以用： 复制代码 音量下 + O 屏幕会出现： 复制代码 File Name to Write: test.txt 按： 复制代码 Enter 文件就保存了。 4️⃣ 退出 nano 按： 复制代码 Ctrl + X 或 复制代码 音量下 + X 就退出编辑器。 5️⃣ 查看文件 在 Termux 输入： Bash 复制代码 cat test.txt 输出： 复制代码 123 你好 Termux ✅ nano最常用3个快捷键 功能 按键 保存 Ctrl + O 退出 Ctrl + X 删除一行 Ctrl + K 如果你愿意，我可以再教你一个 Termux 写脚本的神操作： 不用 nano，直接 一条命令生成 Node.js 脚本，写代码速度会快很多。🚀
